Hoisting

Hoisting is the concept of variable declaration(s) being moved "up" to avoid issues.

console.log(age); // undefined
var age;

What happens here, is that JS "hoists" variable declarations (using var) and sends them to the top of the scope in which they are declared in an attempt to avoid issues.
